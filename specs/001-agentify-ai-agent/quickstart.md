# Quick Start Guide: Agentify - AI Agent 开发调试平台

## 概述

本指南将帮助您快速开始使用 Agentify 平台创建和调试您的第一个 AI Agent。Agentify 是一个可视化工具，允许您通过零代码或低代码方式开发和测试私人 AI Agent 助手。

## 前置条件

- 现代浏览器（Chrome, Firefox, Safari, Edge）
- 无需安装，直接在浏览器中运行

## 快速开始步骤

### 1. 创建新的 AI Agent

1. 打开 Agentify 平台
2. 点击 "新建 Agent" 按钮
3. 输入 Agent 名称和描述
4. 点击 "创建" 完成创建

### 2. 在 Develop 工作空间查看 Agent 分析

1. 在左侧导航栏选择 "Develop" 工作空间
2. 选择您刚创建的 Agent
3. 查看分析数据，包括：
   - Token 消耗量
   - 插件使用情况
   - 工作流数量

### 3. 在 Playground 工作空间编辑提示词并对话

1. 在左侧导航栏选择 "Playground" 工作空间
2. 选择您的 Agent
3. 在提示词编辑器中编辑系统提示词
4. 在对话面板中与 Agent 进行对话测试

### 4. 在 Explore 工作空间创建插件或工作流

1. 在左侧导航栏选择 "Explore" 工作空间
2. 选择 "插件" 或 "工作流" 标签
3. 点击 "新建" 创建新的插件或工作流
4. 配置插件参数或工作流步骤
5. 保存并应用到您的 Agent

## 验证场景

### 场景 1: Agent 创建和基本配置

**Given** 用户首次访问 Agentify 平台
**When** 用户创建一个新的 AI Agent 并配置基本信息
**Then** 系统应成功创建 Agent 并在列表中显示

### 场景 2: 提示词编辑和对话测试

**Given** 用户在 Playground 工作空间
**When** 用户编辑 Agent 提示词并与其对话
**Then** 系统应实时响应并显示对话结果

### 场景 3: 插件和工作流创建

**Given** 用户在 Explore 工作空间
**When** 用户创建新的插件或工作流
**Then** 系统应提供创建向导并保存新创建的组件

## 故障排除

### 问题 1: 页面加载缓慢
- 检查网络连接
- 清除浏览器缓存后重新加载

### 问题 2: 对话无响应
- 检查提示词是否正确配置
- 刷新页面后重试

### 问题 3: 无法保存配置
- 检查浏览器是否支持本地存储
- 确认浏览器未处于无痕模式
